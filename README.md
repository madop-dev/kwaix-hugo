# KWAIX Hugo + Decap CMS

Team von KI-Spezialisten im Norden.

## Quick Start

1. ZIP entpacken
2. `git init && git add . && git commit -m "init"`
3. Push zu GitHub
4. Netlify: New site from Git
5. Decap CMS verbinden
6. Admin: `/admin/` öffnen

## Decap Setup

Nach Netlify Deploy:
1. Geh zu `/admin/`
2. "Login with Netlify"
3. Content editieren

## Struktur

- `admin/` — Decap CMS
- `content/` — Markdown Seiten
- `layouts/` — HTML Templates
- `static/css/` — Styling
